单例模式（饿汉式） -> JDK中的Runtime

简单工厂模式（静态工厂模式）-> JDK中的Calendar
工厂方法模式
抽象工厂模式

原型模式 -> Spring中Bean的scope

建造者模式 -> JDK中的StringBuilder

适配器模式 -> SpringMvc中的handlerAdapter

桥接模式 -> JDBC驱动桥接

装饰者模式 -> JDK的IO体系中

组合模式 -> JDK中的HashMap

外观模式（门面模式）-> MyBatis的Configuration

享元模式 -> JDK中的Integer，String，连接池，线程池

代理模式 -> 静态代理，JDK中的动态代理(InvocationHandler, Proxy.newInstance)，CGLib

模板方法模式 -> RedisTemplate，JDBCTemplate

命令模式 -> JDBCTemplate，(exec, rollback)

状态模式 -> JDK中的线程状态

责任链模式 -> Tomcat中的FilterChain，SpringMVC中的Handler
